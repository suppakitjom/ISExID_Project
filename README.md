# ISExID Project - SpyKids Interactive Flashcard Game

### Hardware
- RC522 RFID Card Reader
- Pushbuttons
- 3D Printed Case
- 7-inch LCD Screen
- Arduino Uno
  
### Software
- PyQt6 App
- Pyserial Communication
